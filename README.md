Fungsi Program
Program ini dibuat untuk membantu seorang siswa RPL SMK Telkom Malang menghitung nilai faktorial dari sebuah angka bulat positif. Dengan program ini, siswa tidak perlu menghitung faktorial secara manual, yang bisa melelahkan dan memakan waktu kalau angkanya besar. Jadi, program ini mempermudah proses perhitungan sekaligus membantu siswa memahami cara kerja logika dan perulangan dalam pemrograman Java.

Cara Kerja Program
Cara kerja programnya cukup sederhana. Pertama, program akan meminta pengguna memasukkan sebuah angka bulat positif. Setelah itu, program akan menghitung faktorial dari angka tersebut dengan cara mengalikan angka itu dengan semua angka di bawahnya sampai angka 1. Proses ini dilakukan secara otomatis menggunakan perulangan. Setelah selesai, program akan menampilkan hasilnya, sehingga pengguna langsung tahu nilai faktorial dari angka yang dimasukkan tanpa harus menghitung satu per satu.
